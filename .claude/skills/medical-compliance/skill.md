# 의료광고법 컴플라이언스 검증

## 법적 근거

- 의료법 제56조 (의료광고의 금지 등)
- 의료법 제57조 (벌칙)
- 의료법 시행령 제23조
- 상세 조문 해석: `references/medical-ad-law-reference.md` 참조

## 8개 위반 카테고리

### 1. 과대광고 (CRITICAL)
치료 효과를 보장하거나 과장하는 표현.
- 금지: "100% 완치", "확실한 효과", "반드시 좋아집니다", "놀라운 결과"
- 허용: "개인차가 있을 수 있습니다", "전문의 상담을 권장합니다"

### 2. 비교광고 (CRITICAL)
다른 의료기관과 비교하거나 최상급 표현 사용.
- 금지: "최고의", "유일한", "가장 많은", "OO보다 나은"
- 허용: "풍부한 경험", "다양한 사례"

### 3. 체험기 오용 (WARNING)
특정 환자 사례를 일반적 결과인 것처럼 서술.
- 금지: "A님도 3회 만에 완치되었습니다" (→ 모든 환자에게 동일한 인상)
- 허용: "개인 경험이며, 결과는 개인에 따라 다를 수 있습니다" 고지 병행

### 4. 미인증 시술/기기 (CRITICAL)
식약처 미인증 의료기기·시술 홍보.
- 검증 방법: 시술/기기명을 식약처 인증 목록과 대조 (가능한 경우)
- 판단 불확실 시: WARNING으로 분류, 사용자 확인 요청

### 5. 가격 오인 유도 (WARNING)
할인율, 이벤트 가격으로 오인을 유도.
- 금지: "50% 할인", "이벤트 특가" (근거 없는 할인율)
- 허용: 구체적 가격 명시 + 부가세/추가비용 고지

### 6. 전후 사진 (CRITICAL)
시술 전후 비교 사진의 부적절한 사용.
- 금지: 보정/편집된 전후 사진, 타인 사진 사용
- AI 이미지 프롬프트에서도 전후 비교 연출 지시 금지

### 7. 자격 과장 (WARNING)
의료진 자격·경력의 과장 표현.
- 금지: "대한민국 최고 전문의", 미검증 수상 이력
- 허용: 객관적 자격 사항 (전문의 자격, 학회 소속)

### 8. 보장 표현 (CRITICAL)
결과를 보장하는 표현.
- 금지: "무조건", "반드시", "걱정 없는", "안전한" (절대적 표현)
- 허용: "최선을 다하겠습니다", "꼼꼼하게 진행합니다"

## 3중 방어 실행 절차

### 1차 방어: 규칙 기반 스캔
`references/prohibited-expressions.md`의 금지 표현 목록과 정규식 매칭.
- 정확 매칭 + 유사 표현 패턴 (예: "100%" → "백프로", "완벽한", "확실한")
- 위치 정보 포함 (파일명:라인번호)

### 2차 방어: LLM 문맥 판단
규칙으로 잡히지 않는 우회 표현을 LLM이 판단한다.
- "자연스럽게 좋아진다" → 문맥상 치료 보장인지 일반 건강 조언인지 판단
- 긴 문단의 전체 맥락을 고려하여 위반 여부 결정

### 3차 방어: 자동 수정 + 재검증
- 각 위반에 대해 구체적 수정안 제시
- 수정 후 1차+2차 방어를 다시 실행하여 수정이 새로운 위반을 만들지 않았는지 확인
