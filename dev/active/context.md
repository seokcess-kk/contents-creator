# 프로젝트 컨텍스트 — Contents Creator

> 최종 갱신: 2026-03-31

## 프로젝트 정체성
- **한 줄 소개**: 네이버 상위 노출 콘텐츠의 패턴을 분석하고, 패턴 기반으로 클라이언트별 차별화된 블로그 콘텐츠를 생성하는 내부 도구
- **핵심 차별점**: "레퍼런스 분석 → 패턴 추출 → 패턴 기반 생성" (AI 추측이 아닌 데이터 기반)
- **용도**: 내부 사용 (수익화 배제)
- **초기 업종**: 의료(한의원, 피부과, 치과) · 뷰티 · 건강

## 핵심 설계 원칙
1. 레퍼런스는 인사이트, 구조는 자체 결정
2. 변이가 모든 층위에서 (5개 층위)
3. 의료법은 1순위 (3중 방어, 타협 없음)
4. 분석 1번, 생성 N번 (패턴 카드 = 자산)
5. 프로필이 차별화의 핵심 (같은 패턴 + 다른 프로필 = 다른 글)

## 현재 상태
- Phase 0 완료: 인프라 부트스트랩 (하네스, 스키마, CLAUDE.md)
- Phase 1 대기: MVP 핵심 파이프라인 구현 시작 전
