## 클라이언트 협상 포인트

1. **상 43개에 단가 집중** — SOV 점유율 임팩트 가장 큼. 그중 **최상 난이도 13개**(검색량 5,000+)는 별도 가중치 적용 권장
2. **위젯 점유 키워드 사전 안내** — BMI계산하기·비만도는 1위 노출에도 계산기 박스가 상단을 점유. 클릭 슬롯 제한 가능성을 클라이언트에 미리 공지
3. **검색량 0 18개는 제외 또는 재발굴 권장** — 노출 보장의 의미 자체가 성립하지 않음. 정직성 어필 포인트
4. **정보형 vs 카테고리 vs 지역+업종 사이의 작업 난이도 차이** — 동일 검색량이어도 정보형(인플루언서·VIEW 풀)이 카테고리(의료 광고)보다 진입 비용 더 높을 수 있음. 단가 세분화 시 활용
5. **분기별 등급 재평가** — 시장 변화 반영, 데이터 기반 단가 정당성 backup
